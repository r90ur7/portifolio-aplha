# AGENTS.md — Portifólio

This is a portfolio project built with pure HTML, CSS, and JavaScript (no frameworks).

## Tech Stack (decided 2026-06-17)

- **HTML5** — Semantic markup (`<header>`, `<nav>`, `<main>`, `<section>`, `<footer>`)
- **CSS3** — Custom Properties, Flexbox, Grid, `@keyframes`, Media Queries, transitions
- **JavaScript (Vanilla)** — DOM API, Intersection Observer, form validation, mobile nav
- **Google Fonts** — Playfair Display (headings) + Inter (body)
- **Formspree** — Contact form backend
- **GitHub Pages** — Hosting via `gh-pages` branch

## Design System

CSS variables in `:root`:
- `--purple`: #7B2D8E (primary)
- `--pink`: #FF69B4 (details)
- `--gold`: #FFD700 (premium accents)
- `--blue`: #4A90D9 (animations)
- `--terracotta`: #E2725B (backgrounds)
- `--orange`: #FF8C00 (CTAs)

## Conventions

- Mobile-first responsive (320px → 1920px)
- Duck SVG icons throughout (nickname: "vírus do pato")
- Dark mode by default
- No external dependencies (no npm, no frameworks)
- Project data fetched from GitHub API topic filter "project"
